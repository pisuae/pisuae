# Project Summary
The PIS UAE Marketplace is an innovative web application that facilitates the buying and selling of diverse products, including electronics, clothing, and smart home devices. Operating on a commission-based vendor model, it enhances user engagement with features such as advanced analytics, a robust review system, and a well-organized product catalog. The platform supports multiple payment methods, including Cash on Delivery (COD) and Stripe, and integrates AI-driven customer support for improved user interactions. Recent enhancements include a real-time order tracking feature and a comprehensive PWA Installation Guide for iOS, aimed at boosting user satisfaction and retention.

# Project Module Description
The project consists of two primary modules: **Frontend** and **Backend**.

## Frontend
- **Pages**: Contains product listings, detailed views, shopping cart, user orders, vendor signup/login, vendor dashboard, analytics, product categories, and the PWA Installation Guide.
- **Components**: Reusable UI elements like buttons, cards, headers, forms, a star rating system, a customer support chatbot, and a search bar.
- **State Management**: Utilizes React hooks for efficient state management with enhanced error handling.
- **Checkout Page**: Now requires full name, email, mobile number, and shipping address for both Cash on Delivery and Stripe payment options.
- **Authentication Flow**: Features a sign-up/login banner on the homepage with improved error handling for DNS/network issues.
- **Navigation Enhancements**: Includes a sticky navigation bar, smooth scrolling, a back-to-top button, and links to the PWA Installation Guide.
- **Search History Feature**: Saves user queries as clickable chips in the search bar.
- **Trending Searches Feature**: Displays popular search queries fetched from the backend.
- **Real-Time Order Tracking**: New feature showing order status updates with timestamps on the Orders page.

## Backend
- **API Endpoints**: Manages CRUD operations for products, orders, vendors, reviews, user preferences, trending searches, and order tracking, with robust error handling.
- **Database Models**: Defines schemas for products, orders, user authentication, vendor profiles, reviews, search history, and order tracking events.

# Technology Stack
- **Frontend**: React, TypeScript, Vite, Tailwind CSS, Recharts, date-fns, react-day-picker
- **Backend**: Python, FastAPI, SQLAlchemy, Alembic
- **Database**: PostgreSQL

# Usage
1. **Install Dependencies**:
   - For the backend: Navigate to `app/backend` and run `pip install -r requirements.txt`.
   - For the frontend: Navigate to `app/frontend` and run `pnpm install`.
   
2. **Build**:
   - For the frontend: Run `pnpm run build` in the `app/frontend` directory.

3. **Run**:
   - Start the backend server by executing `python main.py` in the `app/backend` directory.
   - Start the frontend application using `pnpm run dev` in the `app/frontend` directory.
