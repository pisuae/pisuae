# Project Summary
The PIS UAE Marketplace is an innovative web application designed for tech enthusiasts to buy and sell a wide range of tech components and electronics, including new, refurbished, and used items. It offers a user-friendly interface for easy product exploration and competitive pricing. Vendors can list their items on a commission basis, enhancing the marketplace's offerings. The platform features a vendor analytics section that provides insights into sales performance and visitor engagement, empowering vendors to make informed decisions.

# Project Module Description
The project consists of two primary modules: **Frontend** and **Backend**.

## Frontend
- **Pages**: Includes product listings, product details, shopping cart, user orders, vendor signup/login, vendor dashboard, vendor analytics, and a user profile page featuring a recent orders summary and a spending summary widget.
- **Components**: Reusable UI elements such as buttons, cards, headers, and forms, including an image upload component for vendor product listings.
- **State Management**: Utilizes React hooks for effective state and side effect management.

## Backend
- **API Endpoints**: Handles CRUD operations for products, orders, vendors, and cart items.
- **Database Models**: Defines schemas for products, orders, user authentication, vendor profiles, user preferences, and product views.

# File Description Inventory
- **backend/**: Contains the backend application code, including API routes, models, and database migrations.
- **frontend/**: Contains the frontend application code, including pages, components, and styles.
- **README.md**: Documentation for both frontend and backend.
- **todo.md**: Development tasks and design guidelines for vendor features.
- **image.ts**: Utility for resolving and uploading product images.
- **VendorAnalytics.tsx**: Enhanced page for displaying vendor sales analytics with new chart types, metrics, and data export functionality.
- **Profile.tsx**: Updated page now includes a **Spending Summary Widget** displaying total spent, number of valid orders, and average order value.
- **index.html**: Main HTML file for the frontend application.
- **site.config.json**: Configuration file for the frontend, now includes Google Analytics 4 measurement ID.

# Technology Stack
- **Frontend**: React, TypeScript, Vite, Tailwind CSS, Recharts
- **Backend**: Python, FastAPI, SQLAlchemy, Alembic
- **Database**: PostgreSQL

# Usage
1. **Install Dependencies**:
   - For the backend: Navigate to `app/backend` and run `pip install -r requirements.txt`.
   - For the frontend: Navigate to `app/frontend` and run `pnpm install`.
   
2. **Build**:
   - For the frontend: Run `pnpm run build` in the `app/frontend` directory.

3. **Run**:
   - Start the backend server by executing `python main.py` in the `app/backend` directory.
   - Start the frontend application using `pnpm run dev` in the `app/frontend` directory.
