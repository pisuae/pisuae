# Project Summary
The PIS UAE Marketplace is an innovative web application tailored for tech enthusiasts to buy and sell a diverse array of tech components and electronics, encompassing new, refurbished, and used items. It features a user-friendly interface that simplifies product exploration and competitive pricing. Vendors can list items on a commission basis, enhancing the marketplace's offerings. The platform also includes a vendor analytics section that provides insights into sales performance and visitor engagement, assisting vendors in informed decision-making.

# Project Module Description
The project consists of two primary modules: **Frontend** and **Backend**.

## Frontend
- **Pages**: Includes product listings, product details, shopping cart, user orders, vendor signup/login, vendor dashboard, vendor analytics, and a user profile page featuring a recent orders summary.
- **Components**: Reusable UI elements such as buttons, cards, headers, and forms, including an image upload component for vendor product listings.
- **State Management**: Utilizes React hooks for effective state and side effect management.

## Backend
- **API Endpoints**: Handles CRUD operations for products, orders, vendors, and cart items.
- **Database Models**: Defines schemas for products, orders, user authentication, vendor profiles, user preferences, and product views.

# Technology Stack
- **Frontend**: React, TypeScript, Vite, Tailwind CSS, Recharts
- **Backend**: Python, FastAPI, SQLAlchemy, Alembic
- **Database**: PostgreSQL

# Usage
1. **Install Dependencies**:
   - For the backend: Navigate to `app/backend` and run `pip install -r requirements.txt`.
   - For the frontend: Navigate to `app/frontend` and run `pnpm install`.
   
2. **Build**:
   - For the frontend: Run `pnpm run build` in the `app/frontend` directory.

3. **Run**:
   - Start the backend server by executing `python main.py` in the `app/backend` directory.
   - Start the frontend application using `pnpm run dev` in the `app/frontend` directory.
