# Project Summary
The TechParts Marketplace is a web application designed for buying and selling tech components and electronics. It offers a user-friendly platform where customers can browse, filter, and purchase a variety of products, including new, refurbished, and used items, all with a focus on quality assurance and competitive pricing.

# Project Module Description
The project consists of two main modules: **Frontend** and **Backend**. The frontend is built with React and provides the user interface, while the backend is responsible for managing data, authentication, and business logic.

## Frontend
- **Pages**: Includes pages for product listings, product details, shopping cart, user orders, and authentication.
- **Components**: Reusable UI components for buttons, cards, headers, and forms.
- **State Management**: Uses React hooks for managing state and side effects.

## Backend
- **API Endpoints**: Handles CRUD operations for products, orders, and cart items.
- **Database Models**: Defines schemas for products, orders, and user authentication.

# Technology Stack
- **Frontend**: React, TypeScript, Vite, Tailwind CSS
- **Backend**: Python, FastAPI, SQLAlchemy, Alembic
- **Database**: PostgreSQL

# Usage
1. **Install Dependencies**:
   - For the backend: Navigate to `app/backend` and run `pip install -r requirements.txt`.
   - For the frontend: Navigate to `app/frontend` and run `pnpm install`.
   
2. **Build**:
   - For the frontend: Run `pnpm run build` in the `app/frontend` directory.

3. **Run**:
   - Start the backend server by executing `python main.py` in the `app/backend` directory.
   - Start the frontend application using `pnpm run dev` in the `app/frontend` directory.
