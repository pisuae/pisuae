# Project Summary
The PIS UAE Marketplace is a dynamic web application designed to empower tech enthusiasts by facilitating the buying and selling of tech components and electronics. It provides a user-friendly platform for customers to explore a wide variety of products, including new, refurbished, and used items, while ensuring quality assurance and competitive pricing. The marketplace also supports vendors who can sell their products on a commission basis, enhancing the business model by allowing for a diverse range of sellers. Recently, the platform has introduced image upload functionality for vendors, improving the product listing experience.

# Project Module Description
The project consists of two main modules: **Frontend** and **Backend**. The frontend, developed with React, offers an engaging user interface, while the backend manages data, authentication, and business logic through a robust API.

## Frontend
- **Pages**: Includes product listings, product details, shopping cart, user orders, vendor signup/login, and vendor dashboard.
- **Components**: Features reusable UI elements such as buttons, cards, headers, and forms, including new image upload components for vendor product listings.
- **State Management**: Utilizes React hooks for effective state and side effect management.

## Backend
- **API Endpoints**: Handles CRUD operations for products, orders, vendors, and cart items.
- **Database Models**: Establishes schemas for products, orders, user authentication, and vendor profiles.

# Technology Stack
- **Frontend**: React, TypeScript, Vite, Tailwind CSS
- **Backend**: Python, FastAPI, SQLAlchemy, Alembic
- **Database**: PostgreSQL

# Usage
1. **Install Dependencies**:
   - For the backend: Navigate to `app/backend` and run `pip install -r requirements.txt`.
   - For the frontend: Navigate to `app/frontend` and run `pnpm install`.
   
2. **Build**:
   - For the frontend: Run `pnpm run build` in the `app/frontend` directory.

3. **Run**:
   - Start the backend server by executing `python main.py` in the `app/backend` directory.
   - Start the frontend application using `pnpm run dev` in the `app/frontend` directory.
