# Project Summary
The PIS UAE Marketplace is a dynamic web application aimed at facilitating the buying and selling of a wide variety of products, including electronics, clothing, and smart home devices. It operates on a commission-based vendor model and enhances the user experience with advanced analytics tools, a robust review system, and an organized product catalog. Key features include Cash on Delivery (COD) payment options, an online payment gateway via Stripe, and an AI-driven customer support chatbot. The platform supports Progressive Web App (PWA) functionality, allowing users to install it as a native-like app. Recent enhancements include a user-friendly sign-up/login banner and improved navigation features.

# Project Module Description
The project consists of two primary modules: **Frontend** and **Backend**.

## Frontend
- **Pages**: Includes product listings, detailed product views with reviews, shopping cart, user orders, vendor signup/login, vendor dashboard, analytics, and product categories.
- **Components**: Reusable UI elements such as buttons, cards, headers, forms, a star rating system for reviews, a customer support chatbot, and a new search bar in the sticky navigation.
- **State Management**: Utilizes React hooks for effective state management, including a retry mechanism for API calls.
- **Checkout Page**: Supports both Cash on Delivery and Stripe payment options, requiring shipping address and phone number fields.
- **Authentication Flow**: Features a prominent sign-up/login banner for new users on the homepage, encouraging registration before purchasing.
- **Navigation Enhancements**: Introduced a sticky mini navigation bar with smooth scrolling to major sections, a floating back-to-top button, and a search bar for quick product searches.

## Backend
- **API Endpoints**: Manages CRUD operations for products, orders, vendors, reviews, and user preferences, with enhanced error handling.
- **Database Models**: Defines schemas for products, orders, user authentication, vendor profiles, and reviews, including new fields for payment methods.

# Technology Stack
- **Frontend**: React, TypeScript, Vite, Tailwind CSS, Recharts, date-fns, react-day-picker
- **Backend**: Python, FastAPI, SQLAlchemy, Alembic
- **Database**: PostgreSQL

# Usage
1. **Install Dependencies**:
   - For the backend: Navigate to `app/backend` and run `pip install -r requirements.txt`.
   - For the frontend: Navigate to `app/frontend` and run `pnpm install`.
   
2. **Build**:
   - For the frontend: Run `pnpm run build` in the `app/frontend` directory.

3. **Run**:
   - Start the backend server by executing `python main.py` in the `app/backend` directory.
   - Start the frontend application using `pnpm run dev` in the `app/frontend` directory.
