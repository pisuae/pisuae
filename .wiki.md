# Project Summary
The PIS UAE Marketplace is a dynamic web application that empowers tech enthusiasts to buy and sell tech components and electronics. It offers a user-friendly platform for exploring a diverse range of products, including new, refurbished, and used items, with a focus on quality assurance and competitive pricing. Vendors can sell their products on a commission basis, enhancing the marketplace's diversity. Recently, a vendor analytics section was introduced, providing insights into sales performance and visitor engagement.

# Project Module Description
The project consists of two main modules: **Frontend** and **Backend**. The frontend, developed with React, provides an engaging user interface, while the backend manages data, authentication, and business logic through a robust API.

## Frontend
- **Pages**: Includes product listings, product details, shopping cart, user orders, vendor signup/login, vendor dashboard, and a new vendor analytics page.
- **Components**: Features reusable UI elements such as buttons, cards, headers, and forms, including an image upload component for vendor product listings.
- **State Management**: Utilizes React hooks for effective state and side effect management.

## Backend
- **API Endpoints**: Handles CRUD operations for products, orders, vendors, and cart items.
- **Database Models**: Establishes schemas for products, orders, user authentication, vendor profiles, and product views.

# Technology Stack
- **Frontend**: React, TypeScript, Vite, Tailwind CSS, Recharts
- **Backend**: Python, FastAPI, SQLAlchemy, Alembic
- **Database**: PostgreSQL

# Usage
1. **Install Dependencies**:
   - For the backend: Navigate to `app/backend` and run `pip install -r requirements.txt`.
   - For the frontend: Navigate to `app/frontend` and run `pnpm install`.
   
2. **Build**:
   - For the frontend: Run `pnpm run build` in the `app/frontend` directory.

3. **Run**:
   - Start the backend server by executing `python main.py` in the `app/backend` directory.
   - Start the frontend application using `pnpm run dev` in the `app/frontend` directory.
