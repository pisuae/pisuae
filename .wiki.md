# Project Summary
The TechParts Marketplace is an innovative web application focused on facilitating the buying and selling of tech components and electronics. It provides a seamless platform for customers to explore a wide range of products, including new, refurbished, and used items, with a strong emphasis on quality assurance and competitive pricing.

# Project Module Description
The project consists of two main modules: **Frontend** and **Backend**. The frontend, built with React, delivers an engaging user interface, while the backend manages data, authentication, and business logic through a robust API.

## Frontend
- **Pages**: Features product listings, product details, shopping cart, user orders, and authentication.
- **Components**: Includes reusable UI elements such as buttons, cards, headers, and forms.
- **State Management**: Utilizes React hooks for effective state and side effect management.

## Backend
- **API Endpoints**: Manages CRUD operations for products, orders, and cart items.
- **Database Models**: Establishes schemas for products, orders, and user authentication.

# Technology Stack
- **Frontend**: React, TypeScript, Vite, Tailwind CSS
- **Backend**: Python, FastAPI, SQLAlchemy, Alembic
- **Database**: PostgreSQL

# Usage
1. **Install Dependencies**:
   - For the backend: Navigate to `app/backend` and run `pip install -r requirements.txt`.
   - For the frontend: Navigate to `app/frontend` and run `pnpm install`.
   
2. **Build**:
   - For the frontend: Run `pnpm run build` in the `app/frontend` directory.

3. **Run**:
   - Start the backend server by executing `python main.py` in the `app/backend` directory.
   - Start the frontend application using `pnpm run dev` in the `app/frontend` directory.
