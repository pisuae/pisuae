# Project Summary
The PIS UAE Marketplace is a sophisticated web application tailored for the buying and selling of diverse products, including electronics, clothing, and smart home devices. It employs a commission-based vendor model, enhancing user experience through features like advanced analytics, a comprehensive review system, and a well-structured product catalog. Key functionalities include Cash on Delivery (COD) payment options, an online payment gateway via Stripe, and AI-driven customer support. The platform supports Progressive Web App (PWA) capabilities for a seamless app-like experience. Recent enhancements include a "Trending Searches" feature and an updated PWA Installation Guide for iOS, both aimed at boosting user engagement.

# Project Module Description
The project consists of two main modules: **Frontend** and **Backend**.

## Frontend
- **Pages**: Contains product listings, detailed views, shopping cart, user orders, vendor signup/login, vendor dashboard, analytics, product categories, and the PWA Installation Guide.
- **Components**: Reusable UI elements such as buttons, cards, headers, forms, a star rating system, a customer support chatbot, and a search bar.
- **State Management**: Utilizes React hooks for efficient state management, including a retry mechanism for API calls.
- **Checkout Page**: Supports both Cash on Delivery and Stripe payment options, requiring shipping address and phone number.
- **Authentication Flow**: Features a sign-up/login banner on the homepage.
- **Navigation Enhancements**: Incorporates a sticky navigation bar, smooth scrolling, a back-to-top button, and links to the PWA Installation Guide.
- **Search History Feature**: Saves user queries as clickable chips in the search bar.
- **Trending Searches Feature**: Displays popular search queries fetched from the backend.

## Backend
- **API Endpoints**: Manages CRUD operations for products, orders, vendors, reviews, user preferences, and trending searches, with improved error handling.
- **Database Models**: Defines schemas for products, orders, user authentication, vendor profiles, reviews, and search history.

# Technology Stack
- **Frontend**: React, TypeScript, Vite, Tailwind CSS, Recharts, date-fns, react-day-picker
- **Backend**: Python, FastAPI, SQLAlchemy, Alembic
- **Database**: PostgreSQL

# Usage
1. **Install Dependencies**:
   - For the backend: Navigate to `app/backend` and run `pip install -r requirements.txt`.
   - For the frontend: Navigate to `app/frontend` and run `pnpm install`.
   
2. **Build**:
   - For the frontend: Run `pnpm run build` in the `app/frontend` directory.

3. **Run**:
   - Start the backend server by executing `python main.py` in the `app/backend` directory.
   - Start the frontend application using `pnpm run dev` in the `app/frontend` directory.
